## How to get started

### Prerequisites

- Node.js: https://nodejs.org/en/download/
- npm
  ```sh
  npm install npm@latest -g
  ```
- yarn
  ```sh
  npm install --global yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/astro3012/currency-converter.git
   ```
2. Install packages
   ```sh
   npm install
   ```
   ```sh
   yarn install
   ```
3. Run the client
   ```sh
   npm run start
   ```
   ```sh
   yarn start
   ```
